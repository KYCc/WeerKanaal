# Het weerkanaal Belgie

**Dagelijke reels die het weer van morgen tonen** 

## Projects

### Backend

.NET project dat dagelijks videos produceert

### Frontend

React project dat de frames voorziet

## Dagelijkse stappen

- Fetch weather data
- Launch Playwright
- Inject weather data
- Record page
- Pass frames to FFmpeg
- Add song based on weather
- Output mp4
- Upload to Instagram
